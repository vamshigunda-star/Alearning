Key Improvements Summary

🗂️

Tab-First Navigation

Athletes and Groups are separated into dedicated tabs, eliminating the jumbled single-screen layout and creating clear mental models.

☑️

Multi-Select + Context Actions

Checkboxes enable selecting athletes before choosing an action. The floating "Add to Group" bar appears only when needed, reducing clutter.

🔍

Search & Filter

A search bar in both tabs allows instant filtering. No more endless scrolling to find an athlete or group.

🛡️

Safe Destructive Actions

Swipe-to-delete with confirmation replaces raw "✕" buttons. Inside groups, remove actions show a confirmation dialog.

📌

Single Clear Entry Point

One "+" icon in the header (quick-add athlete) and one FAB per tab. No more ambiguous unlabeled icons or confusing pill buttons.

👥

Expandable Group Detail

Tap a group card to expand inline. See members, drag-to-reorder, and manage — all without navigating to a separate screen.
